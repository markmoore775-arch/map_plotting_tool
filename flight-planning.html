Transfer mission to SD card

Copy your exported KMZ file onto an SD card and import it in DJI Pilot 2.

  1. Click Export KMZ and save the file to your computer.
  2. Copy the .kmz file onto an SD card (root or a folder such as DJI_Missions).
  3. Insert the SD card into your controller.
  4. In DJI Pilot 2, go to Flight Route → Import Route (KMZ/KML).
  5. Browse to the KMZ file and select it.
  6. The mission will appear in your mission library. Open it to review and fly.

Note: Use FAT32 formatting if the controller has trouble reading the SD card.
