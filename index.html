      AirPlot Help

      AirPlot lets you plan and visualise drone flights on an interactive map. Draw flight areas, mark no-fly zones, and place key locations such as take-off and landing points.

      Adding Points

      Add points either via the form in the sidebar or by clicking the pin button on the map toolbar, then choose a type and click the map.

      Location formats supported: UK postcode, decimal lat/lng, Ordnance Survey grid reference, degrees/minutes/seconds (DMS), and What3Words (requires API key in Settings).

      Point Types

      Address/Reference, Primary TOLA, Secondary TOLA, Custom TOLA, Emergency Landing Zone, No-Fly Marker, Hazard, Waypoint, Custom Point, House, and TOLA. Custom points can use a 1–2 character symbol.

      Managing Points

      Use the search box to filter points. Click a point in the list to pan to it and highlight it. Use the edit and delete buttons, or right-click a marker on the map. Drag markers to move them. Click a marker to see a quick-edit popup with label, elevation, and links to View Details or Edit. Double-click a marker or use “View Details” for coordinates in multiple formats (Decimal, DMS, OS Grid Ref, What3Words, Elevation) with copy-to-clipboard. Use Copy All in the Point Details modal to copy all coordinate formats at once.

      Elevation

      Points can display elevation (metres AMSL) when a Mapbox token is configured. Use the Always display elevation on map checkbox when adding or editing a point to show elevation in the quick-edit popup and point list.

      Drawing Shapes

      Right-click anywhere on the map (or long-press on touch devices) to open the context menu. From there you can draw:

      • Circle - click centre, then edge
      • Rectangle - drag corners
      • Polygon - click vertices, double-click to finish
      • Line - click start and end points
      • Arrow - click tail, then tip
      • Flight Path - multi-point path with direction arrows
      • Text - add text labels on the map

      Editing Shapes

      Double-click a shape to open the edit modal. Click a shape to select and drag it. For arrows, select the arrow then drag its handles to rotate or resize. Use Edit Vertices from the context menu (or the vertices button in the sidebar) to adjust polygon/line points. For flight paths and lines, use the sidebar buttons to edit without clicking the narrow path. Measurement labels on circles and flight paths can be dragged to reposition them.

      Copy & Paste

      Right-click a shape to copy it (Ctrl+C when selected), then right-click elsewhere to paste (Ctrl+V). Press Esc to deselect.

      Grid Overlay

      Add a labelled grid over your map area for reference (e.g. search patterns, sector planning). Click the grid button on the map toolbar or choose Grid Overlay from the right-click context menu, then draw a rectangle to define the grid area. Set rows (1–50) and columns (1–26). Columns are labelled A–Z, rows 1–50. Right-click a grid cell to Edit Grid (change rows/columns or visibility) or Clear Grid.

      Drawing Style

      The floating panel shows colour, fill opacity, stroke width, stroke style, and (for flight paths) arrow head size. These apply to new shapes.

      Flight Overview

      For flight paths and lines, use Flight Overview to see altitude statistics. Click the altitude icon (▲) in the sidebar next to a flight path or line, or use the Flight Overview button when editing a shape. You can also right-click the path and choose Flight Overview. A modal shows start and end altitude, highest and lowest points, total variation, net change, total gain and loss, and average change per segment. A waypoint table lists each point with its coordinates, altitude (metres AMSL), and change from the previous point. For lines, start and end points are used. Elevation data requires a Mapbox token in js/config.js.

      Flight Path Distance

      Flight paths can show total distance on the map. Use the Show distance on map checkbox when editing a flight path, or enable Show flight path distance on map in Settings to apply to all flight paths by default.

      Bulk Import

      Import many points at once via Upload CSV, Paste Data (tabular), or Paste List (one location per line). Map columns to your data fields. UK postcodes are supported with bulk lookup.

      Export

      Export points as CSV, or points and shapes as KML for use in other mapping tools. Screenshot Map captures the current map view as an image.

      Save & Load Project

      Save your entire project (points, shapes, settings) as a JSON file. Load it later to restore your work.

      Undo

      Use the Undo button in the sidebar or map toolbar, or Ctrl+Z (Cmd+Z on Mac) to revert the last action.

      Map Toolbar

      The toolbar (top-left) includes: Search - open the Search Location modal; Hand/Pan - drag to move the map; Undo - revert last action; Drop Point - choose type and click map to add a point; drawing tools (circle, rectangle, polygon, line, arrow, flight path); and Grid Overlay - draw a rectangle to add a labelled grid.

      Search Location

      Click the search button on the map toolbar to open the Search Location modal. Enter a postcode, lat/long, OS Grid ref, DMS, or What3Words address. The map will pan and zoom to that location.

      Map Layers

      Use the layer control (top-right) to switch between OpenStreetMap, Topographic, Satellite, and optionally Mapbox or Ordnance Survey (OS Maps requires an API key in Settings).

      UK Airspace

      The airspace key (bottom-right) shows all restriction types. Use Select all to show or hide all types, or tick individual types. Click the collapse button (▼) to minimise the panel.

      • Prohibited - No flying permitted
      • Restricted - Flying limited under certain conditions
      • Danger - Hazardous activities may occur
      • FRZ / Aerodrome - Flight Restriction Zones around protected aerodromes
      • NOTAM - Temporary restrictions from UK NOTAM Archive. Click the arrow (▼) for options: max radius (5-50 NM), drone-relevant only, and opacity. No API key required.
      • RA(T) - Restricted Area Temporary from BGA Airspace Service. Requires username and password in Settings (register at airspace.bgaladder.net).

      Data is sourced from UK AIP ENR 5.1 / NATS UAS. Click any zone for details including vertical limits and activation times. Use the refresh button (↻) to reload airspace data.

      To use real NATS data, see docs/AIRSPACE_DATA.md for download and conversion instructions.

      Geolocation

      Use the locate button to show your current position on the map.

      Sidebar

      Collapse the sidebar (‹ button) for more map space. Use Fit All to zoom the map to include all points.

      Settings

      Configure optional API keys (What3Words, OS Maps, BGA Airspace for RA(T)), and toggle display of point labels, shape measurements, shape labels, and flight path distance.

      Keyboard Shortcuts

      Escape - close modals, exit drawing/drop-point mode, deselect shapes. Ctrl+Z (Cmd+Z) - undo last action. Ctrl+C / Ctrl+V - copy and paste shapes.

      Data Sources

      • UK Postcodes - postcodes.io (free, no API key required)
      • What3Words - What3Words API (requires API key)
      • Ordnance Survey Maps - OS Data Hub (requires API key)
      • OpenStreetMap - OpenStreetMap contributors (map tiles)
      • OpenTopoMap - OpenTopoMap (topographic tiles, SRTM elevation)
      • Mapbox - Mapbox (map tiles, Terrain-RGB elevation; requires token in config)
      • UK Airspace - UK AIP ENR 5.1 / NATS UAS (Prohibited, Restricted, Danger, FRZ)
      • UK NOTAM - UK NOTAM Archive (NATS AIS Contingency PIB, hourly updated)
      • RA(T) Airspace - BGA Airspace Service (requires username/password)

      Libraries & Tools

      • Leaflet 1.9.4 - leafletjs.com - interactive maps
      • Leaflet-Geoman 2.16.0 - geoman.io - drawing and editing shapes
      • Leaflet.Locate 0.79.0 - GitHub - geolocation control
      • Leaflet.Curve 0.9.2 - GitHub - bezier curves
      • Lucide Icons 0.468.0 - lucide.dev - icons
      • html2canvas 1.4.1 - html2canvas - map screenshots
      • Google Fonts - fonts.google.com - Inter font

      Coordinate Systems

      OS Grid Reference conversion uses Ordnance Survey National Grid (OSGB36) to WGS84 transformation (Airy 1830 ellipsoid).
